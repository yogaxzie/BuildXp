❓ FAQ

Apa itu buildXp? ▼
buildXp adalah platform untuk membuat dan deploy website instan menggunakan HTML/CSS/JS atau Python Flask. Cocok untuk prototyping cepat atau hosting project kecil.
Berapa lama website saya aktif? ▼
Free trial aktif selama 7 hari. Untuk VIP member, website aktif selama membership kamu masih valid.
Apakah support database? ▼
Untuk project Python, kamu bisa menggunakan SQLite. Kami tidak menyediakan database server eksternal untuk saat ini.
Bagaimana cara upgrade ke VIP? ▼
Hubungi admin melalui form customer service atau langsung login dan pilih upgrade di dashboard. Admin akan mengaktifkan VIP kamu.
Apakah bisa custom domain? ▼
Untuk saat ini kami hanya menyediakan subdomain dari buildxp.app. Custom domain akan hadir di update mendatang.
